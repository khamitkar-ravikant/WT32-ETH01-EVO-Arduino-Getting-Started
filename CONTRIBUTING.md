# Contributing to WT32-ETH01-EVO-Arduino

Thank you for your interest in contributing.

This repository aims to provide reliable, well-tested examples for the WT32-ETH01-EVO development board based on the ESP32-C3 and DM9051 SPI Ethernet controller.

## How to Contribute

Contributions are welcome in the following areas:

* Bug fixes
* Example projects
* Documentation improvements
* Additional Ethernet features
* HTTPS examples
* MQTT examples
* RS485/Modbus examples
* OTA firmware updates
* Performance improvements

## Before Submitting

Please ensure that:

* The code compiles using Arduino IDE 2.3.x or newer.
* The example has been tested on actual WT32-ETH01-EVO hardware.
* The code is well formatted and documented.
* Serial output is included in the example where applicable.

## Pull Requests

When submitting a Pull Request, please include:

* Description of the change
* Arduino IDE version
* ESP32 Board Package version
* Hardware used
* Serial monitor output (if applicable)
* Screenshots or photographs when useful

## Reporting Issues

When opening an issue, please include:

* Board revision
* Arduino IDE version
* ESP32 Board Package version
* Complete compiler output
* Serial monitor output
* Wiring details
* Steps to reproduce the issue

## Coding Style

* Use descriptive variable names.
* Keep functions focused on a single task.
* Add comments for hardware-specific code.
* Avoid unnecessary delays where possible.

## Future Contributions

Contributions for the following are especially encouraged:

* Static IP examples
* HTTPS client/server examples
* MQTT communication
* Industrial IoT applications
* Modbus RTU/TCP
* Secure TLS communication
* Ethernet OTA
* Home Assistant integration

Thank you for helping improve the WT32-ETH01-EVO Arduino ecosystem.
