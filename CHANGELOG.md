# Changelog

All notable changes to this project will be documented in this file.

The format is based on **Keep a Changelog** and follows **Semantic Versioning**.

---

## [1.0.0] - 2026-06-25

### Added

* Initial public release.
* Ethernet initialization for WT32-ETH01-EVO.
* DM9051 SPI Ethernet support.
* DHCP configuration example.
* Static IP configuration example.
* Ethernet link detection.
* MAC address display.
* IP address display.
* Gateway and DNS information.
* Internet connectivity test (HTTP 204).
* NTP time synchronization example.
* Arduino IDE 2.3.x compatibility.
* ESP32 Board Package 3.3.x compatibility.

### Tested

* WT32-ETH01-EVO
* ESP32-C3 Dev Module
* Arduino IDE 2.3.10
* ESP32 Board Package 3.3.0
